# ALEX Code Agent (npm wrapper)

This package provides an installer for the `alex-code` CLI tool.

The `alex-code` tool is a lightweight AI code agent built in Go. This npm package is a wrapper around the pre-compiled Go binary that downloads the appropriate binary for your platform.

For more information about the `alex-code` agent, please see the main project [README](https://github.com/cklxx/Alex-Code).

## Installation

```bash
npm install -g alex-code
```

## Usage

```bash
alex-code "Your prompt here"
```
