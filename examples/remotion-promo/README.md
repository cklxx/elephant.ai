# Remotion Promo Demo

Generate a demo promotional video for this repository.

## Usage

```bash
cd examples/remotion-promo
npm install
npm run render
```

Output:

- `artifacts/remotion/elephant-promo-demo.mp4`

Render the homepage production asset:

```bash
npm run render:web
```

Output:

- `web/public/media/elephant-home-demo.mp4`

Optional preview studio:

```bash
npm run studio
```
