Why elephant.ai?

💬

Channel-Native

First-class participant in Lark groups, WeChat chats, CLI, and web dashboard. No app-switching, no context loss.

🧠

Persistent Memory

Remembers conversations, decisions, and context across sessions. Auto-saves Lark and WeChat messages. Retrieves relevant history before you ask.

🚀

Autonomous Execution

ReAct loop runs Think-Act-Observe to completion: web search, code execution, file ops, browser automation — without hand-holding.

📚

Built-in Skills

Markdown-driven workflows for deep research, meeting notes, email drafting, slide decks, video production, and code review.

🌐

Multi-Provider LLM

Seamless integration with OpenAI, Claude, DeepSeek, ARK, Ollama, and more. Switch models per chat or globally.

🛡

Approval Gates

Proactivity requires safety. Risky tool calls need explicit human approval via CLI, web, or chat. Full audit trail.

Clean port/adapter boundaries. One runtime powers every surface.
