## Vision

### 10x Check
The 10x version is the **proactive colleague**: a brain that not only responds intelligently but initiates conversations based on signals (CI failures, cost spikes, stale PRs), runs autonomous workflows end-to-end ("ship the auth refactor" → code, test, review, PR, deploy, canary, report), and learns the user's preferences over time. The three-mode brain is the prerequisite architecture — the brain needs to be smart before it can be proactive.

## Rollback Gates

Each phase ships behind `ConversationProcessEnabled`. Per-phase criteria:
- **Any phase**: If routing accuracy drops below 90% → revert prompt/code for that phase
- **Phase 3**: If think-mode p95 latency exceeds 20s → disable think mode (route to delegate)
- **Steady state**: If think-mode rate exceeds 40% for >1 hour → auto-disable think mode
- **Cost ceiling**: Think mode should not exceed 25% of total brain calls in steady state
