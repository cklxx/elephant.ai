ALEX

Terminal-native AI programming agent

npm install -g alex-code

Philosophy

保持简洁清晰，如无需求勿增实体

Built with hexagonal architecture and clean separation of concerns. Focus on essential complexity only.

Features

Dual Interface

Interactive chat mode and streaming command mode

Built-in Tools

File operations, shell, search, git, web, task management

Multi-model

OpenAI, DeepSeek, OpenRouter, Ollama support

Session Persistence

Save and resume conversations with context

Usage

  • alex — Interactive mode
  • alex "task" — Command mode
  • alex -r session_id -i — Resume session
