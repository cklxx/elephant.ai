# Mermaid assets (offline)

This directory vendors Mermaid.js for **offline** rendering inside the `diagram_render` tool.

- Mermaid version: `10.9.0`
- Source: `https://unpkg.com/mermaid@10.9.0/dist/mermaid.min.js`
- License: MIT (Mermaid project)

## Update
1) Download the pinned version:
   - `curl -fsSL https://unpkg.com/mermaid@10.9.0/dist/mermaid.min.js -o internal/tools/builtin/diagram/assets/mermaid.min.js`
2) If bumping versions, update this README accordingly and ensure `diagram_render` still works in sandbox (no external network).
