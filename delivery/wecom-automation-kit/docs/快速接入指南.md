# 企业微信自动化工具包：快速接入指南

## 目标
收到意向金后，30 分钟内完成基础接入，启动 7 天 MVP。

## 客户需提前准备的权限与信息清单
1. **企业微信管理权限**（建议超管或具备应用管理权限账号）
2. **企业ID（CorpID）**
3. **应用 Secret / Token / EncodingAESKey**（用于 webhook 签名验证与解密）
4. **回调 IP 白名单配置权限**（如客户侧启用）
5. **客户数据导出权限**（含字段：最近互动时间、历史订单、复购周期）
6. **标签管理权限**（创建与更新客户标签）
7. **消息发送权限**（应用可发送消息给客户/成员）

## 30 分钟对接流程（建议）
- **0-5 分钟：** 权限核验与参数收集（CorpID、Token、EncodingAESKey）
- **5-15 分钟：** 部署 `scripts/wecom-webhook-receiver.py`，完成回调地址配置
- **15-20 分钟：** 导入 `templates/client-data-import.xlsx` 模板并校验字段
- **20-25 分钟：** 应用 `configs/auto-tag-rules.yaml` 标签规则
- **25-30 分钟：** 回调验签测试 + 首轮样本触达测试（售后回访/到期提醒/沉默激活）

## 验收标准
- webhook 验签通过（GET 回调校验成功）
- POST 回调可正常解密并记录消息
- 标签规则三类场景可触发（售后回访、到期提醒、沉默激活）
- 模板数据可导入并可用于分层触发

## 交付目录
- `scripts/`：接收与验签脚本
- `templates/`：客户数据导入模板
- `configs/`：自动标签规则
- `docs/`：接入与操作说明
