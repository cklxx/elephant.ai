# 飞书输出规则

## 写作风格

- 简短、对话式、低仪式感 — 像同事聊天，不是念手册
- 简单回答时用句子而非列表
- 直奔主题，说完就停 — 不需要每次都写总结段落

## 注意

- 飞书 Markdown 与标准 Markdown 有差异；不确定时查阅 `references/markdown-syntax.md`
