### 研究主题
微信生态「MCP Server + 私域自动化」轻SaaS变现机会。

### 关键发现（web_search）
- MCP（Model Context Protocol）正在从“开发者工具协议”外溢到业务自动化场景，价值点是：把 AI 调用工具的集成成本标准化。
- 微信生态商业化内容持续强调：AI 驱动的私域运营、客服自动化、可审计交付（SLA/合规）可直接映射付费能力。
- 可行切入：以“企业微信私域跟进/客服/预约”作为首批模板场景，用 MCP 统一接入不同工具（CRM、表单、知识库、报表）。

### 新增机会（评分：8.7/10）
**企业微信「MCP 私域自动化连接器 + 托管优化」微SaaS**

- 机会定义：提供标准化 MCP Server（连接企微会话/标签/任务系统）+ 3 套可开箱 SOP（5分钟首响、48h二次唤醒、预约确认），按“订阅 + 托管 + 分成”收费。
- 目标客群：20-200 人、已有企微私域但系统割裂/跟进效率低的团队。

#### MVP（14天）
1) 交付 1 个 MCP 连接器（企微 + CRM/表单二选一）；
2) 上线 3 条自动化 SOP（首响、二次唤醒、预约确认）；
3) 日报看板（SLA达成率、48h回复率、预约率、人工接管率）；
4) 每72小时一次 A/B 迭代（话术与触达时段）。

#### 定价建议（人民币）
- 启动部署费：6999-15999 元；
- 月订阅/托管：3999-8999 元；
- 成果分成：新增回款 5%-12%。

#### 首周验证标准
- >=2 个意向金客户（每个 >=999 元）；
- 5分钟SLA达成率 >=85%；
- 48h回复率提升 >=15%；
- 至少 1 个客户形成前后对比案例。

#### 风险与对策
- 证据质量参差：下一轮优先补官方平台/财报/咨询报告交叉验证；
- 指标归因争议：统一客户后台口径 + 日报留痕；
- 合规风险：敏感词分级 + 频控 + 人工抽检。
