# Security Policy

We take security seriously. Please report vulnerabilities responsibly.

## Reporting a vulnerability

Email: `security@cklxx.dev` (preferred)

Include:
- A clear description of the issue and impact.
- Reproduction steps or proof-of-concept.
- Any relevant logs or screenshots.

We will acknowledge receipt within 72 hours and provide a timeline for fixes.

## Supported versions

Only the latest `main` branch is supported with security updates.
